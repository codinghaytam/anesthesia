    \begin{abstract}
        \noindent
        This work presents four reinforcement learning approaches for automated propofol dosing during surgery, using BIS monitoring and PK/PD simulations. The algorithms range from tabular methods (Q-Learning, Dynamic Programming) to deep RL (DQN), all operating on a fuzzy state representation of BIS error dynamics. Our findings indicate that all models require additional training to achieve clinically acceptable performance, with Q-Learning showing the most severe under-training and DP-based methods demonstrating moderate control instability.
    \end{abstract}

\section{Introduction}

Maintenance of adequate anesthesia depth during surgical procedures represents a critical challenge in modern medicine. The Bispectral Index (BIS) provides a validated measure of hypnotic state, enabling closed-loop control of anesthetic delivery. This work employs reinforcement learning to optimize propofol infusion rates, with the objective of maintaining BIS at a target value of 50 throughout surgery.

The Schnider model provides pharmacokinetic and pharmacodynamic simulations that capture patient-specific drug response characteristics. Four distinct reinforcement learning algorithms are implemented and evaluated: tabular Q-Learning, Dynamic Programming Policy Iteration, Dynamic Programming Value Iteration, and Deep Q-Network (DQN).

\section{State Representation and Actions}

All algorithms share a common fuzzy state representation derived from BIS measurements. The state vector comprises two components:

\begin{equation}
    s = (\text{BIS\_ERROR}, \text{DELTA\_BIS})
\end{equation}

where BIS\_ERROR represents the difference between current BIS and target BIS ($\text{BIS} - \text{BIS}_{\text{target}} \in [-50, +50]$), and DELTA\_BIS captures the temporal derivative ($\text{BIS}[t] - \text{BIS}[t-1] \in [-30, +30]$).

Each continuous value undergoes fuzzification using three membership functions (negative, zero, positive), yielding six fuzzy features. Discretized across 10 bins per feature, this creates a state space of $10^6$ discrete states.

The action space consists of seven discrete propofol infusion rates:

\begin{equation}
    \mathcal{A} = \{0.0, 0.5, 1.0, 2.0, 3.0, 4.0, 6.0\} \ \text{ml/min}
\end{equation}

\section{Reinforcement Learning Models}

\subsection{Q-Learning (Tabular)}

Q-Learning is a model-free, off-policy algorithm that learns optimal action-values through temporal difference updates. The agent maintains a Q-table of size $10^6 \times 7$ and selects actions using an $\epsilon$-greedy policy.

    ]
    
    \node (bis) [box] {$\text{BIS}[t]$};
    \node (compute) [box, right=of bis] {Compute\\BIS\_ERROR,\\DELTA\_BIS};
    \node (fuzzy) [box, right=of compute] {Fuzzy\\State $s$};
    \node (qtable) [box, below=of fuzzy] {Q-Table\\$(10^6 \times 7)$};
    \node (selection) [box, below=of qtable] {$\epsilon$-greedy\\Action Selection};
    \node (reward) [box, right=of selection] {$r = -|\text{BIS\_ERROR}|$\\$- 0.5 \times |\text{DELTA\_BIS}|$};
    \node (action) [box, below=of selection] {Action $a$};
    
    \draw[arrow] (bis) -- (compute);
    \draw[arrow] (compute) -- (fuzzy);

The reward function penalizes both absolute BIS error and oscillation:

\begin{equation}
    r = -|\text{BIS\_ERROR}| - 0.5 \times |\text{DELTA\_BIS}|
\end{equation}

\textbf{Hyperparameters:} $\text{EPISODES} = 1{,}000{,}000$, $\text{STEPS} = 120$, $\alpha = 0.2$, $\gamma = 0.69$, $\epsilon = 0.1$

\textbf{Training Status:} Requires additional training. Current performance indicates insufficient propofol delivery, with MDPE remaining at approximately 94.79\%.

\subsection{Dynamic Programming --- Policy Iteration}

Policy Iteration combines policy evaluation with iterative policy improvement. The algorithm alternates between computing state-action values via SARS' transitions and extracting the greedy policy.

\begin{figure}[htbp]
    \centering
    \begin{tikzpicture}[
        node distance=1cm,
        box/.style={rectangle, draw, minimum width=3cm, minimum height=0.8cm, align=center},
        arrow/.style={->, thick},
        >=stealth
    ]
    
    \node (start) {};
    \node (loop) [box, below=of start] {For each episode:\\BIS $\rightarrow$ Fuzzy State $\rightarrow$ Q-Update};
    \node (update) [box, below=of loop] {$Q[s,a] \leftarrow Q[s,a] + \alpha[r + \gamma \max Q[s',:] - Q[s,a]]$};
    \node (extract) [box, below=of update] {$\pi(s) = \arg\max_a Q(s,a)$};
    \node (end) [below=of extract] {};
    
    \draw[arrow] (start) -- (loop);
    \draw[arrow] (loop) -- (update);

\subsection{Dynamic Programming --- Value Iteration}

Value Iteration computes optimal values directly via the Bellman optimality equation on a precomputed transition matrix. This approach assumes knowledge of the environment dynamics.

\begin{figure}[htbp]
    \centering
    \begin{tikzpicture}[
        node distance=1cm,
        box/.style={rectangle, draw, minimum width=3.5cm, minimum height=0.8cm, align=center},
        arrow/.style={->, thick},
        >=stealth
    ]
    
    \node (transition) [box] {Compute: $P[s,a] = s'$, $R[s,a]$};
    \node (iteration) [box, below=of transition] {Repeat until convergence:\\$Q[s,a] = R[s,a] + \gamma V[P[s,a]]$\\$V[s] = \max_a Q[s,a]$};
    \node (policy) [box, below=of iteration] {$\pi(s) = \arg\max_a (R[s,a] + \gamma V[P[s,a]])$};
    
    \draw[arrow] (transition) -- (iteration);
    \draw[arrow] (iteration) -- (policy);
    
    \end{tikzpicture}
    \caption{Value Iteration algorithm structure}
    \label{fig:value_iter}
\end{figure}

\section{Conclusions}

All evaluated models require additional training to achieve clinically acceptable anesthesia control:

\begin{itemize}
    \item \textbf{Q-Learning:} Severely undertrained. The high MDPE of 94.79\% indicates BIS values remain near baseline (95), suggesting insufficient propofol delivery. The exploration-exploitation balance may require adjustment.
    
    \item \textbf{DP-Policy/Value:} Moderate training achieved. High wobble (21.71\%) demonstrates control instability with oscillating BIS values. The policy appears to struggle with maintaining steady-state control.
    
    \item \textbf{DQN:} Moderate training achieved. Performance mirrors DP-based methods, suggesting the neural network has not yet learned robust control policies.
\end{itemize}

\subsection{Recommended Next Steps}

\begin{enumerate}
    \item Increase training episodes significantly for all algorithms
    \item Tune the reward function to encourage faster BIS reduction
    \item Adjust hyperparameters (learning rate, discount factor, exploration rate)
    \item Consider reward shaping to accelerate initial learning
    \item Implement curriculum learning strategies
\end{enumerate}
